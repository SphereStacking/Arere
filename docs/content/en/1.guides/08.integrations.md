# Terminal Multiplexer Integration

By combining arere with tmux or Zellij, you can build more efficient workflows.

## tmux Integration

### Basic Usage

Save the following script as `arere-tmux.sh` and run it:

```bash
#!/bin/bash
# tmux integration for arere

SESSION_NAME="arere"

# Create new tmux session with 'arere' in the main pane
# and split the window vertically (70% for the bottom pane)
tmux new-session -s $SESSION_NAME -n main \
  'arere' \; \
  split-window -v -p 70 \; \
  select-pane -t 0 \; \
  bind-key Tab select-pane -t :.+
```

This starts a tmux session with the following layout:

```
┌────────────────────────────┐
│ arere                      │  ← arere stays running
│ > git-commit               │
│   docker-up                │
├────────────────────────────┤
│ $ git status               │  ← Regular shell
│ On branch main             │
│ $ █                        │
└────────────────────────────┘
```

### Adding to tmux.conf (Optional)

Add the following to `~/.tmux.conf` to switch panes with Tab key:

```bash
bind-key Tab select-pane -t :.+
```

### Features

- **Top pane**: arere stays running, allowing you to select and execute actions
- **Bottom pane**: For regular shell work (70% of space)
- **Tab key**: Quick pane switching

### Session Management

```bash
# Attach to session
tmux attach -t arere

# Terminate session
tmux kill-session -t arere
```

## Zellij Integration

### Basic Usage

Save the following layout file as `arere.kdl`:

```kdl
// Zellij layout configuration for arere
layout {
    pane split_direction="vertical" {
        // arere running in 40% of the window (left side)
        pane command="arere" size="40%"

        // Regular shell in 60% of the window (right side)
        pane
    }
}
```

Launch command:

```bash
zellij --layout arere.kdl
```

This starts Zellij with the following layout:

```
┌───────────────┬───────────────┐
│ arere         │ $ git status  │
│ > commit      │ On branch main│
│   build       │ $ █           │
│   deploy      │               │
└───────────────┴───────────────┘
```

### Apply Layout to Existing Session

```bash
# Create new tab and apply layout
zellij action new-tab --layout arere.kdl
```

### Features

- **Left pane**: arere stays running, allowing you to select and execute actions (40% of space)
- **Right pane**: For regular shell work (60% of space)
- **Alt + Arrow keys**: Switch between panes (Zellij default)

### Customization

To customize the layout, edit `arere.kdl`:

```kdl
layout {
    pane split_direction="horizontal" {  // Change to horizontal split
        pane command="arere" size="20%"    // Change size to 20%
        pane
    }
}
```

### Add to Zellij Config

Add the following to `~/.config/zellij/config.kdl` to always start with the arere layout:

```kdl
default_layout "arere"
```

Then place the layout file at `~/.config/zellij/layouts/arere.kdl`.

### Controls

- **Alt + Arrow keys**: Switch between panes
- **Ctrl+t + q**: Exit Zellij
- **Ctrl+t + d**: Detach session
