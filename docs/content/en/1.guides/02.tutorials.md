# Hands-on Tutorials

`arere-plugin-tutorial` is an official tutorial plugin that lets you learn step by step while looking at actual code.

## Installation

```bash
npm install -g arere-plugin-tutorial
```

## Tutorial List

### Level Basic

| # | Tutorial | Content |
|---|---|---|
| 01 | hello-world | Create your first action |
| 02 | output-api | How to use the output API |
| 03 | prompt-basics | Prompt basics |
| 04 | prompt-form | Form input |
| 05 | shell-executor | Execute shell commands |

### Level Intermediate

| # | Tutorial | Content |
|---|---|---|
| 06 | i18n-inline | Inline internationalization |
| 07 | i18n-locales | Locale file management |
| 08 | config-usage | Using configuration |
| 09 | error-handling | Error handling |

### Level Advanced

| # | Tutorial | Content |
|---|---|---|
| 10 | visual-feedback | Visual feedback |
| 11 | timing-control | Timing control |
| 12 | headless-mode | Headless mode |
| 13 | external-sdk | External SDK integration |

## How to Run

```bash
# Launch arere and select a tutorial
arere

# Or run directly
arere run 01-hello-world
arere run 02-output-api
# ...
```
