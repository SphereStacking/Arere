# Publishing

## Create README

```markdown
# arere-plugin-example

Example plugin for [arere](https://github.com/your-org/arere).

## Installation

\`\`\`bash
npm install -g arere-plugin-example
\`\`\`

## Usage

After installation, the plugin is automatically detected when you launch arere.

You can enable/disable the plugin in the Settings screen.

## Actions

- `example-hello` - Display Hello
- `example-greet` - Greet someone

## License

MIT
```

## Build

```bash
npm run build
```

After building, a `dist/` directory is created:

```
dist/
├── index.js
└── index.d.ts
```

## Publish to npm

```bash
# Login to npm (first time only)
npm login

# Version bump
npm version patch  # 0.1.0 -> 0.1.1
npm version minor  # 0.1.0 -> 0.2.0
npm version major  # 0.1.0 -> 1.0.0

# Publish
npm publish --access public
```

## Verify Installation

```bash
# Global installation
npm install -g arere-plugin-example

# Verify in arere
arere
```

## Troubleshooting

### Plugin Not Recognized

```bash
# Check if plugin is installed
npm list -g | grep arere-plugin

# Clear cache
rm -rf ~/.cache/arere
```

### Build Errors

```bash
# Delete node_modules and reinstall
rm -rf node_modules package-lock.json
npm install

# Build
npm run build
```

### Publishing Errors

```bash
# Check if logged in to npm
npm whoami

# Re-login
npm login
```
