# Creating Actions

This guide covers how to create actions.

## Hello World

Let's start with the simplest action.

```typescript
// ~/.arere/actions/hello.ts
import { defineAction } from 'arere'

export default defineAction({
  name: 'hello',
  description: 'Display Hello World',
  async run({ tui }) {
    tui.output.log('Hello, World!')
  }
})
```

Run the action:

```bash
arere
# Select "hello" and press Enter
```

## Action Structure

Actions are defined using the `defineAction()` helper:

```typescript
import { defineAction } from 'arere'

export default defineAction({
  // Metadata
  name: 'action-name',        // Optional: Action name (auto-inferred from filename if omitted)
  description: 'Description',  // Required: Description
  category: 'category',        // Optional: Category
  tags: ['tag1', 'tag2'],      // Optional: Tags

  // Run function
  async run(context) {
    // Action processing
    // context includes tui, $, t, env, cwd, config, etc.
  }
})
```

## Category and Tags

**Category**: Classifies actions into broad groups

```typescript
export default defineAction({
  name: 'git-status',
  category: 'git',  // Categorize the action
  // ...
})
```

**Tags**: More detailed classification or keywords for searching

```typescript
export default defineAction({
  name: 'git-commit',
  description: 'Create a Git commit',
  category: 'git',
  tags: ['git', 'commit', 'vcs'],  // Set multiple tags
  // ...
})
```

Tags are displayed in the UI as `#tag1 #tag2` format and are also used for fuzzy search.

## Best Practices

### Proper Error Handling

```typescript
const result = await $`some-command`
if (result.exitCode !== 0) {
  tui.output.error(`An error occurred: ${result.stderr}`)
  return
}
```

### User Feedback

```typescript
tui.output.info('Processing...')
await longRunningTask()
tui.output.success('Completed')
```

### Confirmation Dialog (Dangerous Operations)

```typescript
const confirmed = await tui.prompt.confirm('Are you sure you want to delete?', { defaultValue: false })
if (!confirmed) return
```
