# settings.json

Configure the application behavior.

## Basic Structure

```json
{
  "logLevel": "info",
  "locale": "en",
  "theme": {
    "primaryColor": "green"
  },
  "plugins": {
    "arere-plugin-git": true
  },
  "actionsDir": ".arere"
}
```

## Configuration Options

### logLevel

Specify the log output level.

```json
{
  "logLevel": "info"
}
```

Available values: `"trace"`, `"debug"`, `"info"`, `"warn"`, `"error"`, `"fatal"`

### locale

Specify the UI language.

```json
{
  "locale": "en"
}
```

Available values: `"ja"` (Japanese), `"en"` (English)

### theme

Specify the UI theme.

```json
{
  "theme": {
    "primaryColor": "cyan"
  }
}
```

Available primaryColor: `"green"`, `"cyan"`, `"magenta"`, `"yellow"`, `"blue"`, `"red"`, `"white"`

### ui

Customize UI behavior.

```json
{
  "ui": {
    "bookmarkIcon": "üîñ"
  }
}
```

#### ui.bookmarkIcon

The icon displayed next to bookmarked actions in the list.

Default: `"üîñ"`

Example values: `"üîñ"`, `"‚≠ê"`, `"‚òÖ"`, `"‚ô•"`, `"üìå"`

### bookmarks

Array of bookmarked action IDs. This is automatically managed when you toggle bookmarks with the Tab key.

```json
{
  "bookmarks": [
    "local:my-action",
    "arere-plugin-git:clone-repo"
  ]
}
```

Format: `{source}:{actionName}`
- `local:actionName` - Local actions (project or global)
- `pluginName:actionName` - Plugin actions

### plugins

Specify which plugins to use.

```json
{
  "plugins": {
    "arere-plugin-git": true,
    "arere-plugin-docker": {
      "enabled": true,
      "config": {
        "defaultRegistry": "docker.io"
      }
    },
    "arere-plugin-aws": false
  }
}
```

### actionsDir

Specify the directory where actions are placed.

```json
{
  "actionsDir": "./sys-actions"
}
```

Default: `.arere`

## Merge Behavior

settings.json performs **deep merge**. Only the items specified in workspace settings are overwritten.

**User settings** (`~/.arere/settings.json`):
```json
{
  "logLevel": "debug",
  "locale": "en",
  "plugins": {
    "arere-plugin-git": true
  }
}
```

**Workspace settings** (`.arere/settings.json`):
```json
{
  "logLevel": "info",
  "plugins": {
    "arere-plugin-docker": true
  }
}
```

**Result**:
```json
{
  "logLevel": "info",           // Overwritten by workspace
  "locale": "en",               // Inherited from user settings
  "plugins": {
    "arere-plugin-git": true,   // Inherited from user settings
    "arere-plugin-docker": true // Added by workspace
  }
}
```
