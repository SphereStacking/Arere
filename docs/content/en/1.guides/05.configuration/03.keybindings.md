# keybindings.json

Customize keyboard shortcuts.

## Basic Structure

```json
{
  "global": {
    "exit": ["ctrl+c", "q"],
    "search": ["ctrl+/"],
    "help": ["?"],
    "settings": ["s"],
    "back": ["esc"],
    "confirm": ["enter"]
  },
  "list": {
    "up": ["up"],
    "down": ["down"],
    "select": ["enter"],
    "toggle": ["space"],
    "bookmark": ["tab"]
  }
}
```

## Merge Strategy: Full Replacement

::alert{type="warning"}
keybindings.json uses a **full replacement strategy**. When you define a scope (e.g., `global`), the default settings for that scope are completely replaced.
::

```json
// ❌ Wrong: down, select, toggle, bookmark won't work
{
  "list": {
    "up": ["up", "k"]
  }
}

// ✅ Correct: Define all actions
{
  "list": {
    "up": ["up", "k"],
    "down": ["down", "j"],
    "select": ["enter"],
    "toggle": ["space"],
    "bookmark": ["tab"]
  }
}
```

::alert{type="tip"}
You can omit scopes you don't want to customize. Omitted scopes will use default settings.
::

## Scopes and Actions

### global

| Action | Default | Description |
|--------|---------|-------------|
| `exit` | `ctrl+c`, `q` | Exit the application |
| `search` | `ctrl+/` | Open search mode |
| `help` | `?` | Show help screen |
| `settings` | `s` | Open settings screen |
| `back` | `esc` | Go back to previous screen |
| `confirm` | `enter` | Confirm |

### list

| Action | Default | Description |
|--------|---------|-------------|
| `up` | `↑` | Move to item above |
| `down` | `↓` | Move to item below |
| `select` | `enter` | Select/execute item |
| `toggle` | `space` | Toggle |
| `bookmark` | `tab` | Bookmark/unbookmark action |

### input

| Action | Default | Description |
|--------|---------|-------------|
| `submit` | `enter` | Submit input |
| `cancel` | `esc` | Cancel input |
| `prev` | `←` | Move to previous option |
| `next` | `→` | Move to next option |
| `toggle` | `space` | Toggle |
| `delete` | `backspace`, `delete` | Delete |

### confirm

| Action | Default | Description |
|--------|---------|-------------|
| `yes` | `y`, `Y` | Yes |
| `no` | `n`, `N` | No |

### form

| Action | Default | Description |
|--------|---------|-------------|
| `nextField` | `tab`, `↓` | Move to next field |
| `prevField` | `shift+tab`, `↑` | Move to previous field |
| `submit` | `ctrl+enter` | Submit form |
| `nextStep` | `ctrl+→` | Go to next step |
| `prevStep` | `ctrl+←` | Go to previous step |

### settings

| Action | Default | Description |
|--------|---------|-------------|
| `reset` | `delete` | Reset setting |

## Key Notation

### Basic Keys

| Notation | Key |
|----------|-----|
| `a` - `z` | Alphabet |
| `0` - `9` | Numbers |
| `space` | Space key |
| `enter` | Enter key |
| `esc`, `escape` | Escape key |
| `tab` | Tab key |
| `backspace` | Backspace key |
| `delete` | Delete key |

### Arrow Keys

| Notation | Key |
|----------|-----|
| `up` | ↑ |
| `down` | ↓ |
| `left` | ← |
| `right` | → |

### Modifier Keys

| Notation | Key |
|----------|-----|
| `ctrl+` | Control |
| `shift+` | Shift |
| `meta+`, `cmd+` | Command (macOS) / Windows key |

## Customization Examples

### Vim-style Keybindings

```json
{
  "list": {
    "up": ["up", "k"],
    "down": ["down", "j"],
    "select": ["enter", "l"],
    "toggle": ["space"],
    "bookmark": ["tab"]
  },
  "input": {
    "submit": ["enter"],
    "cancel": ["esc"],
    "prev": ["left", "h"],
    "next": ["right", "l"],
    "toggle": ["space"],
    "delete": ["backspace", "delete"]
  }
}
```

### Adding Search Keys

```json
{
  "global": {
    "exit": ["ctrl+c", "q"],
    "search": ["ctrl+/", "/", "ctrl+f"],
    "help": ["?"],
    "settings": ["s"],
    "back": ["esc"],
    "confirm": ["enter"]
  }
}
```
