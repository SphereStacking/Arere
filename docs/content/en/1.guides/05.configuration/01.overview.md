# Configuration Files

`arere` has two types of configuration files:

| File | Purpose | Merge Strategy |
|------|---------|---------------|
| `settings.json` | Application settings | Deep merge (add/override settings) |
| `keybindings.json` | Keyboard shortcuts | Full replacement (replace by scope) |

## Two-Layer Configuration System

A VSCode-style two-layer configuration system is adopted:

1. **Workspace settings** (project-specific): `./.arere/`
2. **User settings** (global): `~/.arere/`

### Priority

```
Workspace settings > User settings > Default values
```

## Environment Variables

You can specify configuration file locations with environment variables:

```bash
# Specify workspace settings path
export ARERE_WORKSPACE_CONFIG=/path/to/custom/settings.json

# Specify user settings path
export ARERE_USER_CONFIG=/path/to/custom/user-settings.json

arere
```
