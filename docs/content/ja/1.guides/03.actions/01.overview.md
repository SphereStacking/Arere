# アクションとは

アクションは、arere で実行できるタスクの単位です。TypeScript で記述し、ファジー検索で選択して実行します。

## アクションの配置場所

アクションは2つの場所に配置できます：

### グローバルアクション

すべてのプロジェクトで使用できるアクション：

```
~/.arere/actions/
├── git-commit.ts
├── git-switch.ts
└── docker-compose.ts
```

### プロジェクト固有アクション

特定のプロジェクトでのみ使用するアクション：

```
~/my-project/.arere/actions/
├── deploy.ts
└── test-e2e.ts
```

プロジェクトディレクトリで `arere` を実行すると、グローバルとプロジェクトの両方のアクションが表示されます。

## 読み込み優先順位

同名のアクションがある場合、後から読み込まれたものが優先されます（Last-Wins）：

1. **プラグインアクション**（最高優先）
2. **グローバルアクション**
3. **プロジェクトアクション**（最低優先）
