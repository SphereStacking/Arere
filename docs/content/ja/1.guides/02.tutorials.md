# 実践チュートリアル

`arere-plugin-tutorial` は、実際にコードを見ながらステップバイステップで学べる公式チュートリアルプラグインです。

## インストール

```bash
npm install -g arere-plugin-tutorial
```

## チュートリアル一覧

### Level Basic

| # | チュートリアル | 内容 |
|---|---|---|
| 01 | hello-world | 最初のアクション作成 |
| 02 | output-api | 出力APIの使い方 |
| 03 | prompt-basics | プロンプトの基本 |
| 04 | prompt-form | フォーム入力 |
| 05 | shell-executor | シェルコマンド実行 |

### Level Intermediate

| # | チュートリアル | 内容 |
|---|---|---|
| 06 | i18n-inline | インライン国際化 |
| 07 | i18n-locales | ロケールファイル管理 |
| 08 | config-usage | 設定の活用 |
| 09 | error-handling | エラーハンドリング |

### Level Advanced

| # | チュートリアル | 内容 |
|---|---|---|
| 10 | visual-feedback | 視覚的フィードバック |
| 11 | timing-control | タイミング制御 |
| 12 | headless-mode | ヘッドレスモード |
| 13 | external-sdk | 外部SDK連携 |

## 実行方法

```bash
# arere を起動してチュートリアルを選択
arere

# または直接実行
arere run 01-hello-world
arere run 02-output-api
# ...
```
