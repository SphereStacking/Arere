# Getting Started

`arere` は TypeScript ベースの常駐型ターミナルランチャーです。このガイドでは、`arere` の基本的な使い方を学びます。

## インストール

### npm でインストール（推奨）

```bash
npm install -g arere
```

### その他のパッケージマネージャー

```bash
# yarn
yarn global add arere

# pnpm
pnpm add -g arere

# npx（インストール不要、毎回ダウンロード）
npx arere
```

### 必要要件

- Node.js >= 18.0.0
- npm >= 9.0.0
- OS: macOS, Linux, Windows (WSL推奨)

## 最初のアクションを作成

アクションは以下のディレクトリに配置します（必要に応じて作成してください）：

- **グローバルアクション**: `~/.arere/actions/`
- **プロジェクト固有アクション**: `./.arere/actions/`

### 方法1: テンプレートから自動生成（推奨）

`arere-plugin-create` プラグインを使うと、対話形式でアクションを作成できます：

```bash
# プラグインをインストール
npm install -g arere-plugin-create

# arere を起動
arere
```

`create-action` を選択すると、対話形式で以下を設定できます：

1. **アクション名** - ファイル名になります（例: `my-action`）
2. **説明** - アクション一覧に表示される説明
3. **カテゴリ** - utility, development, git, testing, demo から選択
4. **タグ** - 検索用のタグを複数選択
5. **テンプレート** - basic, advanced, with-i18n から選択
6. **保存先** - workspace（プロジェクト）または global（グローバル）

設定が完了すると、自動的にアクションファイルが生成されます。

### 方法2: 手動で作成

テンプレートを使わず、手動でファイルを作成することもできます：

```bash
mkdir -p ~/.arere/actions
```

```typescript
// ~/.arere/actions/hello.ts
import { defineAction } from 'arere'

export default defineAction({
  name: 'hello',
  description: 'Hello World を表示',
  async run({ tui }) {
    const name = await tui.prompt.text('お名前は？')
    tui.output.success(`こんにちは、${name}さん！`)
  }
})
```

## 基本的な使い方

```bash
arere
# または短縮コマンド
arr
```

ファジー検索UIが表示されます：

```
┌─────────────────────────────────────┐
│ 🔍 検索:                            │
│                                     │
│ > 👋 hello - Hello World を表示     │
│                                     │
└─────────────────────────────────────┘
```

入力フィールドに文字を入力すると、ファジー検索でアクションを絞り込めます：

```
検索: git st
→ git-status, git-stash などがヒット
```

### キーボードショートカット

| キー | 操作 |
|------|------|
| `↑` / `↓` | リストを移動 |
| `Enter` | 選択したアクションを実行 |
| `Tab` | アクションをブックマーク/解除 |
| `Ctrl+/` | 検索モードを開く |
| `s` | 設定を開く |
| `?` | ヘルプを表示 |
| `q` / `Ctrl+c` | 終了 |

ブックマークしたアクションには 🔖 アイコンが表示されます。このアイコンは設定でカスタマイズできます。
