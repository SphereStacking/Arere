# settings.json

アプリケーションの動作を設定します。

## 基本構造

```json
{
  "logLevel": "info",
  "locale": "ja",
  "theme": {
    "primaryColor": "green"
  },
  "plugins": {
    "arere-plugin-git": true
  },
  "actionsDir": ".arere"
}
```

## 設定オプション

### logLevel

ログ出力のレベルを指定します。

```json
{
  "logLevel": "info"
}
```

利用可能な値: `"trace"`, `"debug"`, `"info"`, `"warn"`, `"error"`, `"fatal"`

### locale

UIの言語を指定します。

```json
{
  "locale": "ja"
}
```

利用可能な値: `"ja"` (日本語), `"en"` (英語)

### theme

UIのテーマを指定します。

```json
{
  "theme": {
    "primaryColor": "cyan"
  }
}
```

利用可能な primaryColor: `"green"`, `"cyan"`, `"magenta"`, `"yellow"`, `"blue"`, `"red"`, `"white"`

### plugins

使用するプラグインを指定します。

```json
{
  "plugins": {
    "arere-plugin-git": true,
    "arere-plugin-docker": {
      "enabled": true,
      "config": {
        "defaultRegistry": "docker.io"
      }
    },
    "arere-plugin-aws": false
  }
}
```

### actionsDir

アクションを配置するディレクトリを指定します。

```json
{
  "actionsDir": "./sys-actions"
}
```

デフォルト: `.arere`

## マージ動作

settings.json は **深いマージ** を行います。ワークスペース設定で指定した項目のみが上書きされます。

**ユーザー設定** (`~/.arere/settings.json`):
```json
{
  "logLevel": "debug",
  "locale": "ja",
  "plugins": {
    "arere-plugin-git": true
  }
}
```

**ワークスペース設定** (`.arere/settings.json`):
```json
{
  "logLevel": "info",
  "plugins": {
    "arere-plugin-docker": true
  }
}
```

**結果**:
```json
{
  "logLevel": "info",           // ワークスペースで上書き
  "locale": "ja",               // ユーザー設定から継承
  "plugins": {
    "arere-plugin-git": true,   // ユーザー設定から継承
    "arere-plugin-docker": true // ワークスペースで追加
  }
}
```
