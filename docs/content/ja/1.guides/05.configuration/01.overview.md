# 設定ファイルとは

`arere` には2種類の設定ファイルがあります：

| ファイル | 用途 | マージ戦略 |
|----------|------|-----------|
| `settings.json` | アプリケーション設定 | 深いマージ（設定を追加・上書き） |
| `keybindings.json` | キーボードショートカット | 完全置換（スコープ単位で置き換え） |

## 2層設定システム

VSCode 風の2層設定システムを採用しています：

1. **ワークスペース設定** (プロジェクト固有): `./.arere/`
2. **ユーザー設定** (グローバル): `~/.arere/`

### 優先順位

```
ワークスペース設定 > ユーザー設定 > デフォルト値
```

## 環境変数

設定ファイルの場所を環境変数で指定できます：

```bash
# ワークスペース設定のパスを指定
export ARERE_WORKSPACE_CONFIG=/path/to/custom/settings.json

# ユーザー設定のパスを指定
export ARERE_USER_CONFIG=/path/to/custom/user-settings.json

arere
```
