# arere-plugin-timer

> 生産性向上のためのタイマーユーティリティ

日本語 | [English](./README.md)

## 概要

このプラグインは arere ターミナルランチャー用のタイマー機能を提供します。カウントダウンタイマーやポモドーロテクニックで作業に集中しましょう。

## 機能

- **timer**: カスタマイズ可能なシンプルなカウントダウンタイマー
- **pomodoro**: 作業/休憩サイクル付きポモドーロテクニックタイマー
- **サウンド通知**: タイマー完了時にアラーム音
- **i18n対応**: 英語・日本語対応

## インストール

```bash
npm install arere-plugin-timer
```

## 設定

`.arere/settings.json` に追加:

```json
{
  "plugins": {
    "arere-plugin-timer": true
  }
}
```

## アクション

### timer

通知付きシンプルカウントダウンタイマー。

**機能**:
- 柔軟な時間入力: `5m`, `30s`, `1h`, `1h30m`, `90s`
- リアルタイムカウントダウン表示
- 完了時にアラーム音

**使い方**:
1. アクションリストから「timer」を選択
2. 時間を入力（例: 25分なら `25m`）
3. カウントダウン表示でタイマー開始
4. 完了時にアラーム再生

### pomodoro

集中作業セッション用ポモドーロテクニックタイマー。

**デフォルト設定**:
- 作業時間: 25分
- 短い休憩: 5分
- 長い休憩: 15分
- 長い休憩までのサイクル数: 4

**機能**:
- カスタマイズ可能な時間設定
- サイクル進捗の視覚的インジケーター
- 作業/休憩の自動切り替え
- 合計作業時間のセッションサマリー
- フェーズ間のサウンド通知

**使い方**:
1. アクションリストから「pomodoro」を選択
2. 時間を設定（またはデフォルトを使用）
3. サイクルをこなす
4. プロンプトに従って休憩を取る
5. 終了時にセッションサマリーを確認
